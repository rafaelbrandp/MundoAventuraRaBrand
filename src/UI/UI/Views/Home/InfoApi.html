Bienvenido a Mi API REST

Una API moderna desarrollada con .NET Core 3.1 que proporciona endpoints para gestionar recursos de manera eficiente y segura.

Ver Documentación

Endpoints Disponibles

GET /api/usuarios

Obtiene la lista de todos los usuarios registrados en el sistema.

GET /api/usuarios/{id}

Obtiene los detalles de un usuario específico mediante su ID.

POST /api/usuarios

Crea un nuevo usuario en el sistema.

PUT /api/usuarios/{id}

Actualiza la información de un usuario existente.

DELETE /api/usuarios/{id}

Elimina un usuario del sistema.
